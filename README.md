# 双生：遇见另一半的美好：）

> 写下此时的心情匹配和你一样的TA吧~

双生app是一款与众不同的写日记软件，很多朋友写日记坚持不下去是因为太枯燥乏味，没有兴趣感，而这款软件可以在你写日记的同时匹配更多同类型心情的日记主人，让你们一起分享彼此的心情，共建友谊。欢迎下载体验。

* 产品主页：[https://two-life.github.io](https://two-life.github.io)
